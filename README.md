Longyearbyen Ionosonde Watchdog Monitor



  ## Overview

This code serves as an interface between an Arduino board and the computer it is linked to.
The idea is to use an Arduino board to detect changes on the watchdog board created for the Longyearbyen ionosonde. 
The state changes in the ports are then detected and reported to the computer by the serial usb connection and then read by the python code.
The python code prints the messages into the terminal, but also timestamps them and writes them into a log file.


  ## Requirements

This programme in ran on a Linux computer.
The python code relies on the libraries present in requirements.txt


  ## Installation and config

1/ Copy the folder into /opt/ and eventually rename it
2/ Create a python virtual environment in '/opt/Ionosonde_watchdog_monitoring/.venv', and install the required libraries within
3/ Modify the config.ini file :
  - log_path = desired absolute path for the log file that will be generated by the code. By default, it will be in the main folder
  - usb_port = stable port name for your arduino : 
    -- plug your Arduino board
    -- execute 'ls /dev/serial/by-id/' in the terminal
    -- identify your Arduino id
    -- set 'usb_port = /dev/serial/by-id/{your arduino id}'
4/ To test the installation, activate your virtual environment and execute 'python /opt/Ionosonde_watchdog_monitoring/run.py' in the terminal
5/ Create a systemd service that executes run.py with the environment automatically when the computer turns on and activate the service
